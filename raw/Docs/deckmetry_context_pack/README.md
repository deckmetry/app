## Core concept

Deckmetry turns deck inputs into a structured material takeoff, pricing estimate, lead, order request, and project/order workflow.

The system has two primary front-end experiences:
- **Homeowner path**: estimate, material list, project exploration, and labor quote request.
- **Contractor path**: takeoff, contractor-specific pricing, order request, delivery scheduling, and order status tracking.

The system also needs an internal/admin portal for Wehrung's salespeople to review leads, contractor requests, delivery requests, and BOMs before confirming availability and fulfillment.
